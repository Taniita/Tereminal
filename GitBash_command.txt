1. Посмотреть где я : 
pwd

2. Создать папку : 
mkdir group_31

3. Зайти в папку :
cd group_31

4. Создать 3 папки : 
mkdir dir_1 dir_2 dir_3 или mkdir dir{1..3}

5. Зайти любую папку :
cd dir_1

6. Создать 5 файлов (3 txt, 2 json) :
touch 1.txt
touch 2.txt
touch 3.txt
touch 1.json
touch 2.json

7. Создать 3 папки :
mkdir dir_4 dir_5 dir_6

8. Вывести список содержимого :
ls -la

9. Открыть любой txt файл :
vim 1.txt
 
10. Написать туда что-нибудь, любой текст :
1) the first
2) the second 
3) the third 

11. Сохранить и выйти :
esc Shift :wq

12. Выйти из папки на уровень выше :
cd ..

13. Переместить любые 2 файла, которые вы создали, в любую другую папку :
mv 1.txt dir_2/1.txt
mv 1.json dir_3/1.json

14. Скопировать любые 2 файла, которые вы создали, в любую другую папку :
cp 1.txt/dir_3
cp 2.json/dir_3

15. Найти файл по имени :
find "dir_1"

16. Посмотреть содержимое в режиме реального времени :
grep my dir_1

17. Вывести несколько первых строк из текстового файла :
cat 1.txt

18. Вывести несколько последних строк из текстового файла :
cat dir_1/1.txt

19. Посмотреть содержимое длинного файла :
less dir_1/2.txt

20. Вывести дату и время :
date 
time
ls -a
